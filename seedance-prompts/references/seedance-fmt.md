# 即梦 Seedance 2.0 提示词进阶规范

## 镜头句三模块结构

每个镜头一句，三个模块用分号分隔，模块内部不用分号：

```
镜头N：[模块一：主体·动作·台词]；[模块二：场景·光影·音效]；[模块三：运镜·景别·时长]。
```

---

## 模块一：主体·动作·台词

**主体**：写角色名，不写外貌/服装/发型（由参考图控制）。

**动作**：写节拍序列，用"→"连接，具体到身体部位，**禁止情绪词**。

| ❌ 禁止 | ✅ 替换为 |
|--------|---------|
| 她很悲伤 | 她低下头，肩膀轻微下沉 |
| 他很愤怒 | 他的手指收紧，眉头皱紧 |
| 他开心地跑过去 | 他加快脚步，嘴角上扬 |

**台词**：用 `{}` 包裹，嵌入动作描述中间自然位置：

```
弥时背对他继续走，开口说{那就当我说了。}，步伐没有停
```

---

## 模块二：场景·光影·音效

**场景**：空间 + 时间 + 氛围关键词。

**光影**：写光源性质、光位、明暗分布。**严禁色温描述（暖色调/冷蓝调/金色/蓝调等）**。

| ❌ 禁止 | ✅ 替换为 |
|--------|---------|
| 暖色调光线 | 摊位灯泡硬光从右侧斜射 |
| 金色阳光 | 午后斜射阳光从左侧窗户透入，光束清晰可见 |
| 冷蓝调夜景 | 霓虹灯散射软光，积水地面映出碎光 |

**音效**：用 `<>` 包裹：

```
<伞骨挂上棚架的金属摩擦声，伞面"啪"地翻折的脆响>
```

---

## 模块三：运镜·景别·时长

**景别速查**：

| 景别 | 取景范围 |
|------|---------|
| 远景 | 人物极小，环境主导 |
| 全景 | 人物全身入镜 |
| 中景 | 腰部以上 |
| 近景 | 胸部以上 |
| 特写 | 面部或单一物体 |
| 大特写 | 局部放大：眼、手、道具细节 |

**时长**：单镜不超过 10 秒，超过须拆分。

---

## 素材引用规则

**段落 1** — 用图片锚定人物外观与场景：
```
参考<图片>中的[角色名]人物外观，将其中[发型·服装·体态2-3个特征]定义为[角色名]；参考<图片>场景氛围，场景发生在...，真实空间关系是...。
```

**段落 2 及之后** — 素材引用必须以视频延长句开头：
```
<视频>最后一个画面延长[本段秒数]秒视频，保持场景空间一致与角色站位不变。；参考<图片>中的[角色名]...
```

**SC 编号变化（空间切换）时** — 延长句须注明空间差异：
```
<视频>最后一个画面延长[本段秒数]秒视频，保持角色运动方向一致，注意空间切换：从[旧SC名称]进入[新SC名称]，[关键光线/环境差异]。；参考<图片>...
```

**素材占位符**：只写 `<图片>`、`<视频>`、`<音频>`，不带编号。

---

## 分段规则

- 单段总时长不超过 **15 秒**
- X 次切镜 = 本段镜头数 − 1（2个镜头 = 1次切镜）
- 镜头编号跨全部段落**全局连续递增**，不随段落重置

---

## 跳接风险速查

**高风险情形**：同一主体 + 同景别家族 + 同机位角度 + 动作未完成

修正优先级：
1. 可合并且不超过 15 秒 → 合并
2. 改上一镜为反应镜头/道具特写/空镜/背影
3. 改下一镜为换主体/换景别/换角度

---

## 完整段落示例

```
第1段（8秒，1次切镜）：
参考<图片>中的弥时人物外观，将其中黑色雨衣、短发、单肩背包定义为弥时；参考<图片>场景氛围，场景发生在夜晚街市，真实空间关系是露天摊位排成长街，弥时在摊位间穿行。

镜头1：弥时双手撑伞、迈步穿过摊位间的积水路段，伞面在摊灯下一闪而过，开口说{就算你不在了。}，步伐没有停；夜晚街市SC01，摊位灯泡硬光从两侧斜射，积水路面反光，<雨声底噪，伞面拍打声>；跟随运镜，中景景别，5秒。

镜头2：弥时侧脸停在一处摊位前，低头看着摊上的物件，手指轻轻拨动其中一件，没有拿起；同上SC01，摊位灯泡侧光打亮她的侧脸，背景摊位灯光虚化，<轻微拨弄声，雨声持续>；固定机位，近景景别，3秒。
```

## 常见错误速查

| 错误 | 正确做法 |
|------|---------|
| 光影写了色温（暖/冷/金色） | 改为光源性质+光位 |
| 出现情绪词（悲伤/愤怒/绝望） | 改为具体身体动作 |
| 段内镜头编号重置 | 全局连续编号 |
| `<图片1>` `<视频A>` 带编号 | 只写 `<图片>` `<视频>` |
| 段落2+忘写视频延长句 | 素材引用必须以 `<视频>最后一个画面延长X秒...` 开头 |
| 单段超过15秒 | 拆成两段 |
| 单镜超过10秒 | 拆分或缩短 |
| 外貌描述写进镜头句 | 删掉，外貌由参考图控制 |
